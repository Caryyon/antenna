# Code of Conduct

## Our Pledge

We're building software for a small community. Be cool to each other.

## Standards

**Do:**
- Be respectful and constructive
- Welcome newcomers
- Accept feedback gracefully
- Focus on what's best for the project

**Don't:**
- Harass, troll, or insult others
- Share others' private info
- Be a jerk

## Enforcement

If someone's being a problem, reach out to the maintainers. We'll handle it.

## Attribution

This is a simplified code of conduct. For a more comprehensive version, see the [Contributor Covenant](https://www.contributor-covenant.org/).
